Chapter 2. Starting BibleTime

How to start BibleTime

Starting BibleTime from KDE

BibleTime is a KDE program and is integrated with the KDE desktop. You can launch BibleTime from the KDE Start Menu in the Applications group with this icon:

BibleTime can also be launched from a terminal command prompt. To launch BibleTime, open a terminal window and type: 

bibletime

Other window managers

BibleTime can be used with other window managers such as Gnome, BlackBox or Sawfish, providing KDE is already installed on your computer. If your KDE is installed in /opt/kde, then the start-up command in a terminal would be:

export KDEDIR=/opt/kde  #your KDE directory
bibletime

Startup customization

From a terminal you can use BibleTime to open a random verse in the default bible: 

bibletime --open-default-bible "<random>"
To open at a given passage use: 
bibletime --open-default-bible "John 3:16"
You can also use keys in your current bookname language.
