Chapter 4. Configuring BibleTime

In this section you find a description of the different possibilities to configure BibleTime: The Preferences Dialog and the Configure Toolbars dialog.

Preferences Dialog

The BibleTime user interface can be customized in many ways depending on your needs. You can access the configuration dialog by selecting Settings Configure BibleTime.

Startup

The startup behavior can be customized. Select from the following options:

  • Show tip of the day
  • Show startup logo
  • Restore windows from the last BibleTime session

Fonts

By default, BibleTime uses the KDE general display font which can be specified in the KDE Control Center. You can override this font if neccessary. Some languages require special fonts to be displayed correctly, and this dialog allows you to specify a custom font for each language.

The Options dialog - Fonts.

BibleTime can now use all fonts supported by KDE/QT. These may be Bitmap Fonts, Adobe® Type1 Fonts, Adobe® Type3 fonts, or TrueType®.

As long as the modules you are interested in display correctly nothing needs to be done here. If a module only displays as a series of question marks (??????) or empty boxes, then you know that the standard KDE display font does not contain the characters used in this module.

To correct this, choose this module`s language from the drop down menu. Select the use custom font checkbox. Now select a font. For example, a font that supports many languages is Code2000. If no installed font can display the module you are interested in, try installing the KDE localization package for that language.

Installing fonts

Detailed font installation instructions are outside the scope of this handbook. KDE`s KControl Center contains an excellent and easy to use font installation module. Please read its documentation. For further information you might want to refer to the Unicode HOWTO.

Tip

If you use a small font like Clearlyu (about 22kb), BibleTime will run faster than with a large font like Bitstream Cyberbit® (about 12Mb).

Obtaining Fonts

Fonts can be obtained from a number of sources:

  • Your distribution.
  • Your distribution's localization packages.
  • An existing Microsoft®Windows® installation on the same computer.
  • A font collection, such as are available from Adobe® or Bitstream®.
  • Online font collections.

Unicode fonts support more characters than other fonts, and some of these fonts are available at no charge. None of available fonts includes all characters defined in the Unicode standard, so you may want to use different fonts for different languages.

Display templates

Display templates define the rendering of text (colors, size etc.). Various built-in templates are available. If you select one, you will see a preview on the right pane.

Sessions

Sessions define the appearance of the work area, for example which windows are open and which texts should be displayed in these windows. Once a session is created, you can use the menu item Window->Save session to save the current session into an existing session file. Use Window->Load session to load a an existing session into BibleTime. Play around with this helpful feature to see what you can do with it. For example, you might want to create one session with modules that you use for Bible study, and another one for devotional reading. Once you created and saved those sessions, you can switch between them quickly.

Sword

Many features provided by the Sword backend can now be customized in BibleTime. These features are documented right in the dialog.

HotKeys

HotKeys are special key commands that can be used in the place of the menu items and icons. A number of BibleTime's commands have predefined HotKeys (see this section for a complete listing). All of BibleTime's commands can be assigned HotKeys here. This is very helpful to quickly access the functions that you need the most.
