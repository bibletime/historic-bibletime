HotKeys index

This is index of all hotkeys and their corresponding description in the handbook. The hotkeys are sorted (roughly) alphabetical. If you want to directly find out which hotkey a certain menu item has, you can either look at the entry itelf in BibleTime (as it always shows the hotkey), or you can look it up in this section.

HotkeyDescription
Alt+Ctrl+AWindow->Auto cascade equivalent; toggle automatic window cascading.
Alt+Ctrl+SWindow->Save as new session equivalent; saves current layout as new session.
Alt+Ctrl+TWindow->Auto tile equivalent; toggle automatic window tiling.
Alt+Ctrl+WWindow->Close all equivalent; closes all open windows.
F1Help->Handbook equivalent; opens the handbook.
F2Help->BibleStudy Howto equivalent; opens the BibleStudy Howto.
F3Help->Daily tip equivalent; opens the tip of the day.
F4Settings->Configure Sword equivalent; opens the Sword configuration dialog.
F5Window->Configure sessions equivalent; opens the main configuration dialog, where you can configure your sessions.
F8View->Show main index equivalent; toggles display of the main index.
F9View->Show info display equivalent; toggles display of the info display.
Shift+F1Help->What`s this? equivalent; changes the mouse cursor to a help arrow. Click on any element that you want more information about and a tooltip will appear.
Ctrl+AWindow->Cascade equivalent; cascades all open windows.
Ctrl+BMoves a bible or commentary read window to the next book.
Ctrl+CCopies selected text.
Ctrl+FSearch->Search in default bible equivalent; opens the search dialog to search in the default bible.
Ctrl+GSearch->Search in open module(s) equivalent; opens the search dialog to search in all currently opened modules.
Ctrl+HMoves a bible or commentary read window to the next chapter.
Ctrl+TWindow->Tile equivalent; tiles all open windows.
Ctrl+VMoves a read window to the next verse or entry.
Ctrl+Shift+BMoves a bible or commentary read window to the previous book.
Ctrl+Shift+FView->Fullscreen Mode equivalent; toggles fullscreen display mode.
Ctrl+Shift+HMoves a bible or commentary read window to the previous chapter.
Ctrl+Shift+VMoves a read window to the previous verse or entry.
