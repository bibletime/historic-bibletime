HotKeys

HotKeys are special key commands that can be used in the place of the menu items and icons. A number of BibleTime's commands have predefined HotKeys (see this section for a complete listing). All of BibleTime's commands can be assigned HotKeys here. This is very helpful to quickly access the functions that you need the most.
