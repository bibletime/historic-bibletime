Preparing Electronic Texts for the CCEL

Simple Guidelines

Here are a few simple guidelines for preparing an electronic text in Microsoft Word (or another word processor) for easy addition to the Christian Classics Ethereal Library.

  • Enter all of the text in the book just as it appears in the print edition. Formatting should also be similar, but you don't need to maintain line breaks or exact font.
  • Insert footnotes as regular Word footnotes, using Ctrl-Alt-F
  • At the start of a page in the print edition, e.g. page xii, insert the code
      <pb n="xii"/>
    Apart from that code, you can join up pages and delete intervening matter.
  • At the start of each chapter or section that appears in the Table of Contents, insert the code
      <div1 title="Section Title Goes Here">
  • At each scripture reference, e.g. Rom. 8:28, mark as follows:
      <scripRef>Rom. 8:28</scripRef>
Then save the document in RTF format and mail it to me. Your work will make adding the document to the CCEL much easier!
