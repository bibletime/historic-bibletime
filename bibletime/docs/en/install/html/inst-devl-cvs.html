Developer CVS Access via SSH

Only project developers can access the CVS tree via this method. SSH1 must be installed on your client machine. Substitute modulename and developername with the proper values. Enter your site password when prompted.

To checkout a fresh copy of Bibletime

cd $home/your development directory
export CVS_RSH=ssh      # Activation of ssh is needed only once per session
cvs -z3 -d:ext:developername@cvs.bibletime.sourceforge.net:/cvsroot/bibletime co modulename

To update your copy of the Bibletime CVS

cvs -z5 update -Pd

To commit your changes to the CVS repository

cvs -z5 commit -m "comments for this commit"

To add new files that you have created

cvs -z5 add my_new_file
