Compiler notes

The time required to compile bibletime is dependent on your system speed and the amount of RAM installed. If you have a system with lots of RAM compiling can be speeded up by using the make -j2 option. -j allows make to launch two simultaneous jobs.

If your computer is low on RAM and processor, then the best way to speed up compiling is to shut down X and run the compile from a terminal. On a AMD 133 with 32MB ram this saves several hours of compile time.
