5.4. CVS GUI Interface

TkCVS can be used to access the sword and bibletime repositories.

The current version of TkCVS (6.3) has a bug that makes it impossible to log on to a CVS repository after the GUI interface is up. The workaround is to open a terminal, log onto the CVS server, and then to start TkCVS.

 $ export CVSROOT=:pserver:anonymous@cvs.crosswire.org:/cvs/bibletime
 $ cvs login
 $ Password: anonymous
 $ tkcvs
