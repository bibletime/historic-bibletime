Chapter 5. Getting Development Sources

This chapter describes what must be done to obtain components of Bibletime and sword from the CVS repository. The sources will usually successfully compile, however as the CVS may contain unproven code there will be days when the compiles code does not work. This is after all software under development.

If you want stable code, please download one of the released versions of Bibletime from The Bibletime Home Page

5.1. General Information

Bibletime consists of sword, the sword modules, and bibletime. Sword is the backend (something like a librarian) that fetches sword modules and passes them to Bibletime for presentation. All three of these must be installed for bibletime to function.

The sword modules used in the current bibletime development module are the released versions. These are available from Crosswire Bible Society. There are development versions of the sword modules available for some ongoing crosswire projects, these are not used by any version of Bibletime just now. If you are interested in sword development modules please contact <scribe@crosswire.org >

The Bibletime development project is hosted on Sourceforge. Current development information and the latest CVS are available on Sourceforge.
