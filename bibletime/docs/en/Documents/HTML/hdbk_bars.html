Chapter 4. Main toolbar and menubar

4.1. Toolbar

4.1.1. Configuration

The main window toolbar is configurable. Access this with the "Configure tool bars" entry under the "Settings" menubar entry.

Each of the buttons has tooltips to tell what their function is. For buttons with no pixmap, the buttons could appear as blank, or just blank bars.

4.1.2. Button Action
