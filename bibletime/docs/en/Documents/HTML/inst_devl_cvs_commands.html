5.3. CVS Commands

The following commands are used when working with CVS repositories.

To update your source, log on to the CVS repository (if you used the anonymous user account the last time you must use it now as well). Change the current directory of your terminal to your local repository (for example cd ./Develop). Then use the following command:

  cvs -z4 update -Pd

Other commands useful to developers (these commands will not work for anonymous user) are:

 Commit your changes:
         cvs -z4 commit path/of/file
 or
         cd path/of/file
         cvs -z4 commit path/of/file
 
 Add a new file to the CVS repository (for example a new part of the
 handbook):
         cvs -z4 add path/of/file
         cvs -z4 commit
 
 To remove a file:
         cvs -z4 remove path/of/file
         cvs -z4 commit
