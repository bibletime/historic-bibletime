1.3. Installing bibletime

1.3.1. Installing bibletime from RPM

If you install your bibletime from an rpm, open your usual package manager and install the RPM. If you have missed anything package manager will tell you.

1.3.2. Installing bibletime from source.

open a bash terminal, as a regular user, then do the following:

 cd unzipped_source_directory
 #####################################
 # Modify these as discussed in      #
 # setting environment variables     #
 # If the environment is already set,#
 # this is not needed.               #
 #####################################
 export QTDIR=/usr/lib/qt2
 export KDEDIR=/usr
 export KDEDIRS=/usr
 export PATH=$KDEDIRS/bin:$PATH
 export PATH=$QTDIR:$PATH
 export PATH=$KDEDIR:$PATH
 #####################################
 # For CVS users only                #
 # make -f Makefile.cvs              #
 # All other users skip to next line #
 #####################################
 ./configure
 make
 #####################################
 # Switch to super user mode         #
 #####################################
 su
 password ****
 make install
 
 #####################################
 #Launch bibletime                   #
 #####################################
 bibletime

The last command should launch bibletime.

bibletime installs fully integrated with KDE; however it may not operate correctly if the environment variables are not set up correctly. You can add them to /$home/bash_profileto set these environmental variables at startup

You will need to log out and log in for the kde integration to become effective.
