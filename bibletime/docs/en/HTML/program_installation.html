Chapter 1. Installation Information

1.1. Requirements for Bibletime and supporting programs

Bibletime is provided in many forms. We hope that one of these will support your system. At this time we only support linux (and unix when you compile it yourself).

Table 1-1. Bibletime Program File Availability and Requirements

filenameVersionDistributionRequirementsdescription
bibletime-1.0-mandrake8.0.1mdk.i.586.rpm 1.0 Mandrake 8.0.1qt2, KDE2, sword, processor class 586qt2, KDE2 are provided with the distribution
bibletime-1.0.static.1mdk.rpm 1.0 unlisted distributionsany X window manager, linux, intel 586This is the version with as many supporting programs compiled in as possible. This should run on almost any linux with a window manager.
bibletime-1.0.src.tar.gz 1.0 unlisted distributions or unixkde2, qt2, compiler, make, swordThis is the compile it yourself version, it will support any system running kde2 and qt2
