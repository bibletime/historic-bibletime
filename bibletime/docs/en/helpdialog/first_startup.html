The first startup of BibleTime!

This table describes the features and concept of BibleTime!
Feature Description
MDI MDI is the abbrevation for Multiple Document Interface. With the help of MDI it's possibble to have more than one presenter opened at the same time! You may view two different bibles side by side and compare directly their text. But it's also possible to open only one window like in BibleTime 0.2x. The menu Window is used to work with the MDI interface.
The MDI area will only contain presenters, nothing else. This is done in this way because other dialogs would confuse lots of users.
Presenter A presenter is a window inside the MDI area of BibleTime. A presenters presents the content of a SWORD module in a reasonable format (for example Bibles will be displayed as HTML text with hyperlinks to verse references).
We use presenters to have a general approach for the future: SWORD will be extended more and more and new module type will be created. We simply create a new presenter type for a new module type, isn't this easy to understand? BibleTime will be extensible in the future, no matter what types of modules will be created (e.g. maps or sound modules ...).
Groupmanager The groupmanager is the tree on the left side of BibleTime's main window. The groupmanager combines the bookmarkmanagement and modulemanagement of BibleTime 0.2x. To work with the Groupmanager use the right mousebutton menu.
The groupmanager is Drag 'n' Drop capable: You can drag verse references of a presenter into the groupmanager to create a bookmark, and you can move items in the tree (e.g. move groups or bookmarks into another group)
Searchinterface The searchpart is now in an own window. Click with the right mousebutton on a module and select "Search module(s)" to open the searchdialog. The dialog contains sections to enter the searchphrase, set the searchscope and to navigate through the found items. The last tab shows the graphical search analysis.
Darg 'n' Drop Drag&Drop s an important part of BibleTime! You can drag modules, verse references, bookmarks, groups, searchresults etc. Places where you can drop them are the groupmanager, the presenters and the commentary editor.
Simply play around a litle to find out how our Drag&Drop sstem does work! We hope you like it.
