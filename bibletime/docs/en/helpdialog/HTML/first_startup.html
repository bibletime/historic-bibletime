Chapter 1. First Startup

Table 1-1. The first startup of BibleTime! This table describes the features of BibleTime!

FeatureDescription
MDI MDI is the abbreviation for Multiple Document Interface. With the help of MDI it's possible to have more than one presenter opened at the same time! You may view two different bibles side by side and compare their text. But it's also possible to open only one window as in BibleTime 0.2x. The menu Window is used to work with the MDI interface. The MDI area will only contain presenters, nothing else. This clean presentation avoids confusing users with too many choices.
Presenter A presenter is a window inside the MDI area of BibleTime. A presenters presents the content of a SWORD module in a reasonable format (for example Bibles will be displayed as HTML text with hyperlinks to verse references). We use presenters to have a general approach for the future. Crosswire will extend SWORD as more material is made available for distribution. When new module types are created for Sword we will simply create additional presenters. This approach makes BibleTime extensible in the future, and able to display any type of module that might be created in (for example maps or sound modules ...).
Group manager The group manager is the tree on the left side of BibleTime's main window. The group manager combines the bookmark management and module management of BibleTime 0.2x. To work with the Group manager use the right mouse button menu. The group manager is Drag 'n' Drop capable: You can drag verse references from a presenter into the group manager to create a bookmark, and you can move items in the tree (e.g. move groups or bookmarks into another group).
Search interface The search feature is now in its own window. Click with the right mousebutton on a module and select "Search module(s)" to open the searchdialog. The dialog contains sections to enter the searchphrase, set the searchscope and to navigate through the found items. The last tab shows the graphical search analysis .
Drag 'n' Drop Drag & Drop is an important part of BibleTime! You can drag modules, verse references, bookmarks, groups, search results etc. Places where you can drop them are the group manager, the presenters and the commentary editor. Simply play around a little to find out how our Drag & Drop system does work! We hope you like it.

URL Description

www.crosswire.org This is the home of Crosswire! Crosswire is the Bible society which produces the Bible text module library - and the library manager SWORD! You can down load the Bible text modules from this page. Most of the modules available from crosswire are either in the public domain or have been released to crosswire for distribution by the copyright holder. Modules that are not released for free distribution are locked. Please see the helpdialog on locked modules before downloading locked modules.
