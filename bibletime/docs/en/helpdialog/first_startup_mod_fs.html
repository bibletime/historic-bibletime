The first startup of BibleTime!

This table describes the features and concept of BibleTime!
Feature Description
MDI MDI is the abbreviation for Multiple Document Interface. With the help of MDI it's possible to have more than one presenter opened at the same time! You may view two different bibles side by side and directly compare their text. But it's also possible to open only one window as was done in BibleTime 0.2x. The menu Window is used to apply BibleTime features (for example the search dialog) to the contents of presenters in the MDI interface.
The Bibletime team wants to make this software user friendly, so we are making every effort to make Bibletime simple. For this reason the MDI window will only display presenters.
Presenter A presenter is a window inside the MDI area of BibleTime. A presenters presents the content of a SWORD module in a reasonable format (for example Bibles will be displayed as HTML text with hyperlinks to verse references).
We use presenters to have a general approach for the future:
Crosswire will extend SWORD as more material is made available for distribution.
When new module types are created for Sword we will simply create additional presenters.
This approach makes BibleTime extensible in the future, and able to display any type of module that might be created in (for example maps or sound modules ...).
Groupmanager The groupmanager is the tree on the left side of BibleTime's main window. The groupmanager combines the bookmarkmanagement and modulemanagement of BibleTime 0.2x. To work with the Groupmanager use the right mousebutton menu.
The groupmanager is Drag 'n' Drop enabled: You can drag verse references of a presenter into the groupmanager to create a bookmark, and you can move items in the tree (e.g. move groups or bookmarks into another group)
Searchinterface The search feature is now in its own window. Click with the right mousebutton on a module and select "Search module(s)" to open the searchdialog. The dialog contains sections to enter the searchphrase, set the searchscope and to navigate through the found items. The last tab shows the graphical search analysis.
Drag 'n' Drop Drag&Drop is an important part of BibleTime! You can drag modules, verse references, bookmarks, groups, searchresults etc. Places where you can drop them are the groupmanager, the presenters and the commentary editor.
Simply play around a little to find out how our Drag&Drop system works! We hope you like it.
