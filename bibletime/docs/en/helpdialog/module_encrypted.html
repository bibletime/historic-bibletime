Question Answer
What is an encrypted module? An encryptted module is a module which is locked with a key. Only user who have the right key to unlock the module are able to reac it.
How do I unlock a module? For normal users it's sadly not possible at this time to use encrypted modules. The problem is that the copyright owners of the modules do not want to release the stuff together with SWORD.
But I you have a key click with the right mousebutton on the module and select "Unlock module ...". A dialog appears, please enter here the key to unlock the encrypted dmoule.
Why are modules locked? Modules are locked because the copyright owners of the module do not want to release their texts together with the SWORD project.
Please contact the copyright owners and ask for the modules. I you want to pay for it please let them know, this is a thing they like. We know freely available module would be great, so try to get them!
You can also visit www.bibletime.de and read the "Module" section, there is contact information. If your country isn't listed there please create such a page and sent them to us!
Thank you very much for your efforts!
