Output

Exporting

To export text as HTML, text, or to the clipboard from an open document. Do a right mouse click in the document, and a drop down menu will appear. Select the desired option, A save text dialog box will appear for you to enter the desired file name.

Printing

The printing system has been changed, This section is under revusion

Printing from Bibletime is rather basic and is intended as a utility. If you are composing a document or presentation containing text from Bibletime documents, we suggest that you use one of the presentation or editing tools on your system. See the instructions on exporting.

To print, select an open document, then select print. The print dialog box will appear. You will have the option of sending your selection to the printer, or previewing it with kghostview.
