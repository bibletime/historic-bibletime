Menubar

File

New window

opens another instance of Bibletime

Close

closes the presenter that has focus

Quit

Quits Bibletime

Window

Cascade

Cascades open presenters

Tile

Tiles open presenters

Auto Cascade

Automatically cascades newly opened presenters

Auto Tile

ditto for tiling newly opened presenters

Close all

Closes all open presenters

List of open modules

Do I have to explain?

Settings

Show Toolbar

Show or hide the main toolbar, normally shown

Configure Toolbar

show the Configure Toolbar dialog

Preferences

show the Preferences dialog

Show Strongs

display Strong's number in the text (only certain modules use this function)

Show Footnotes

display footnotes in the text (only certain modules use this function)

Show Main Index

show or hide the module list window. Normally it is not hidden, unless you want maximum real estate for displaying modules. BTW, resizing the Main Index window can accomplish the same thing, although to a lesser extent.

Help

Contents

Opens KDE's help browser with Bibletime's handbook

What's This

allows you to click on certain items for contextual help

Bug Report

send a bug report to the program's maintainers to help development

About Bibletime

opens a window with general Bibletime information

About KDE

opens a window with general KDE information
