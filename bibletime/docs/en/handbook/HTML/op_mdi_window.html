4.2. MDI Window

This is where the opened modules are displayed. Modules are displayed in "presenters", that are specific to the three types of modules.

4.2.1. Controlling open modules

Presenters (the windows that contain the modules) can be controlled and manipulated with normal KDE window/titlebar functions. In addition to this, you can tile and cascade open presenters. Use the appropriate toolbar buttons, or menu functions under "Window" in the main menubar. Presenters can be set to automatically cascade or tile also.

4.2.2. Navigating in open modules

Using the comboboxes and presenter toolbar buttons are the way to navigate within a module. You can directly select book, secter, or verse using the comboboxes, and advance or back up secters and verses using the toolbar buttons.

4.2.3. Autoscrolling

One of BibleTime's advanced features is the ability to control more than one open presenter at a time. For Bible and Commentary modules, the rightmost toolbar button is the Autoscroll button. Selecting it in open presenters will allow that module it to scroll in tandem with other modules with autoscrolling on.

4.2.4. Editing your own commentary

The toolbar of commentary presenters has another button, with the picture of a pencil. For the "Personal" commentary module, click that button to be able to enter your own notes into the Personal commentary, or to edit notes you've already entered. When you're done, click on the "Save" (disk) button to save your notes. To delete notes for a selected verse, click the button with the trashcan. You can also set the endpoint of your notes, if for example you want to use the same notes to comment on a range of verses. Click the "Edit" button again to return to normal commentary browsing mode.
