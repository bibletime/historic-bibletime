6.2. First Startup of BibleTime

Table 6-1. The first startup of BibleTime! This table describes the features of BibleTime!

FeatureDescription
MDI MDI is the abbrevation for Multiple Document Interface. With the help of MDI it's possible to have more than one presenter opened at the same time! You may view two different bibles side by side and compare their text. But it's also possible to open only one window as in BibleTime 0.2x. The menu Window is used to work with the MDI interface. The MDI area will only contain presenters, nothing else. This clean presentation avoids confusing users with too many choices.
Presenter A presenter is a window inside the MDI area of BibleTime. A presenters presents the content of a SWORD module in a reasonable format (for example Bibles will be displayed as HTML text with hyperlinks to verse references). We use presenters to have a general approach for the future: SWORD will be extended more and more and new module types will be created. We simply create a new presenter type for a new module types, isn't this easy to understand? BibleTime will be extensible in the future, to include all new modules (e.g. graphics, sound, etc.).
Group manager The group manager is the tree on the left side of BibleTime's main window. The group manager combines the bookmark management and module management of BibleTime 0.2x. To work with the Group manager use the right mouse button menu. The group manager is Drag 'n' Drop capable: You can drag verse references from a presenter into the group manager to create a bookmark, and you can move items in the tree (e.g. move groups or bookmarks into another group)
Search interface The searchpart is now in its own window. Click with the right mouse button on a module and select "Search module(s)" to open the search dialog. The dialog contains sections to enter the search phrase, set the search scope and to navigate through the found items. The last tab shows the graphical search analysis.
Drag 'n' Drop Drag & Drop is an important part of BibleTime! You can drag modules, verse references, bookmarks, groups, search results etc. Places where you can drop them are the group manager, the presenters and the commentary editor. Simply play around a little to find out how our Drag & Drop system does work! We hope you like it.
