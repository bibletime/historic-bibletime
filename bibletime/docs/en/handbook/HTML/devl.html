Chapter 3. Installation of the Development Version

The current development version is available from the crosswire CVS repository. It is also necessary to install the latest version of sword from the CVS repository. On some systems it is also necessary to install the basic libraries of KDE 2 (the package is called kdelibs) and qt-2.2.2 or 2.2.3. It is recommended that an attempt be made to install bibletime and sword with the qt and kde packages that are present in your distribution before downloading the most current kde and qt sources.

3.1. Content Versioning System (CVS)

To download the CVS version of sword, do this:

 [me@localhost $home]export CVSROOT=:pserver:anonymous@cvs.crosswire.org:/cvs/sword
 [me@localhost $home]cd /usr/src
 [me@localhost src]cvs login
 [me@localhost src]  password ******    #use "anonymous" for the password
 [me@localhost src]checkout -z3 checkout sword

Now down load the bibletime development files.

 
 [me@localhost src]export CVSROOT=:pserver:anonymous@cvs.crosswire.org:/cvs/bibletime
 [me@localhost src]cvs login
 [me@localhost src]  password ******   #use "anonymous" for the password
 [me@localhost src]checkout -z3 checkout bibletime-2
 
 exit will close the cvs window.

Now you should have /usr/src/sword and /usr/src/bibletime-2
