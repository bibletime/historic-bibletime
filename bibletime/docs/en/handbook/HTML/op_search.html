4.3. Searching in modules

4.3.1. Accessing search functions

You can search in a module by right-clicking on the module name in the module list and selecting "Search in module(s)". By shift- or ctrl-clicking on other module names you can select more than one. Then follow the same procedure to open the search dialog. You will be searching in all of these modules.

4.3.2. Using the Search dialog

Search Text - In this pane enter the text string to be searched for, and set search properties. "Regular Expression" uses standard regexp syntax to refine the search.

Search Scope - This allows you to define search boundaries. You can have no boundaries, search within the results of the previous search, or set your own boundaries within the modules to be searched. For each module listed as to be searched, you can enter and delete lower and upper boundaries. Remember to click "Add bound" before searching.

Search Result - This tells you first, by module, how many instances of the search string were found. Double-clicking on that module will open the list of references. Clicking on a reference opens that verse up in the window below. Drag the reference and drop it on a module in the main module list to open the module to that verse in a new presenter

Search Analysis - This gives a graphic analysis of the number of instances the search string was found in each book. It can display results for more than one book at a time, and you have the option of saving the graph as an image file on your hard drive.
