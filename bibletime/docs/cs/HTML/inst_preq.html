Chapter 2. Bibletime Prerequisites

Bibletime is still under development. Versions 0.3 or 0.2 are available and relatively stable. The earlier versions use Qt 1.x and kde 1.x, so the required libraries will already be installed.

2.1. Requirements for Bibletime Version 1.0

This document describes how to install BibleTime 1.0 All of the packages required for Bibletime to compile are included in mandrake 7.2 and SuSE 7.1. It is also possible to compile Bibletime on most other distributions. If you encounter compile errors, please read the messages, they should lead you to missing packages which you will probably find on your installation disks.

The following table contains information of the two packages which are new additions to the package set which can be added to most distributions, and are required to enable Bibletime 1.0 to work.

Table 2-1. Packages required for installation of Bibletime 1.0 and later.

PackageLocation
QT 2.2x or later http://www.trolltech.com
KDE 2.x (at least the libraries) http://www.kde.org
