3.2. Installazione dei sorgenti BibleTime

Prima devi scompattare il file tar.gz scaricato (se usate la versione CVS i sorgenti saranno già estratte perciò ignora questo passo) in una directory temporanea. Fai qualcosa come: 

 mkdir /tmp
 cd /tmp
 tar -xzf bibletime.tar.gz
 cd /tmp/bibletime

(il file tar sarà estratto in una directory con un nome simile a bibletime)

Adesso apri un terminale ed esegui i seguenti commandi per compilare i sorgenti.

 $ cd /tmp/bibletime         # Path del sorgente
 $ export QTDIR=/usr/lib/qt2 # directory QT 2.2.x (dove QT 2.2 o ulteriore è installato)
 $ export KDEDIR=/usr/local/kde2    # directory KDE2 (dove KDE2 è installato)
 $ make -f Makefile.cvs             # necessario soltanto per la versione CVS
 $ ./configure                      # crea il makefile
 $ make                             # compila BibleTime
 $ su
 $ password *****
 $ make install                     # installa BibleTime

Il Path a QTDIR e KDE2 può essere diverso nel tuo computer.

Adesso apri un terminale (come utente normale) e scrivi bibletime. BibleTime si avvierà.
