3.5. Output

3.5.1. Esportazione

Per esportare il testo come HTML, testo o negli appunti da un documento aperto, fai un click col tasto destro nel documento, e apparirà un menu a tendina. Seleziona l'opzione desiderata e inserisci il nome del file nella casella che apparirà.

3.5.2. Stampa

Il sistema di stampta è stato modificato. Questa sezione è in fase di revisione.

La stampa in BibleTime offre i servizi di base ed è intesa come utilità. Se stai componendo un documento o una presentazione contenente testo dei documenti di BibleTime, ti consigliamo di usare uno degli strumenti di presentazione o impagianzione del vostro sistema. Consulta le istruzione per l'esportazione.

Per stampare, seleziona un documento aperto, quindi la voce stampa. La fienestra di dialogo di stampa si aprirà. Puoi scegliere se inviare la selezione alla stampante o averne un'anteprima con kghostview.
