Capitolo 5. Ottenere i di sorgenti in sviluppo

5.1. Accesso CVS anonimo

Il CVS SourceForge di questo progetto può essere ricercato tramite CVS anonimo (pserver) seguendo le istruzioni qua sotto. Il modulo che volete ricercare deve essere specificato come nome del modulo. Quando ti viene chiesta una password per anonymous, premi semplicemente il tasto Enter. 

 cvs -d:pserver:anonymous@cvs.bibletime.sourceforge.net:/cvsroot/bibletime login
 cvs -z3 -d:pserver:anonymous@cvs.bibletime.sourceforge.net:/cvsroot/bibletime co nome del modulo

Aggiornamenti dall'interno della directory del modulo non richiedono il parametro -d.
