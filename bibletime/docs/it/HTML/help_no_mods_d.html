5.1. Cos'è la directory mods.d?

La directory mods.d è la directory che dovrebbe contenere i file di configurazione dei moduli. Devi avere questa directory, altrimenti non puoi usare né Bibletime né SWORD! Per favore, leggi le istruzioni di installazione sulla nostra home page.

5.2. Come risolvo questo problema?

La soluzione è creare una directory mods.d: Vai nella directory di SWORD e digita "mkdir mods.d" per crearla. Ora copiaci i file di configurazione dei tuoi moduli e rimuovi (se esiste) il file mods.conf che non è più necessario. La directory mods.d sostituisce questo file.

5.3. Posso avviare Bibletime senza questa directory?

Sì, è possibile. Ma non molto utile. Non ci sono moduli nel gestore dei gruppi. E' possibile testare bibletime in questo modo, ma non lavorare con i moduli.
