2.2. Installazione dei programmi richiesti

Scarica i sorgenti necessari e compila le librerie

Per primi devi compilare i sorgenti delle QT 2, se le QT 2.2 o superiori non sono incluso nel CD della vostra distribuzione. Si presume che i sorgenti delle QT siano in /usr/lib/qt2, quelli di KDE2 in /usr/src/kde2/kdelibs ed la directory di KDE2 sia /usr/local/kde2, se non ti piace usa i tuoi parametri.

 export QTDIR=/usr/lib/qt2
 cd $QTDIR
 ./configure -sm -system-libpng -system-jpeg
 make

Se hai abbastanza RAM e una macchina veloce puoi accelerare il processo di compilazione aggiungendo "-j2" ai parametri di make. Per esempio "make -j2". Se avete una macchina lenta con poca RAM non usate X-window e avviate make in console per accelerare il processo. Dopo che il processo di compilazione è finito devi compilare le librerie necessarie per KDE2. Fai qualcosa di simile:

  export QTDIR=/usr/lib/qt2 export KDEDIR=/usr/local/kde2
 cd /usr/src/kde2/kdelibs
 ./configure
 make
 make install

La compilazione di QT e di KDE dovrebbe terminare senza problemi o errori. È' di nuovo possibile di aggiungere "-j2" per accelerare il processo di compilazione se avete abbastanze RAM e un PC rapido.
