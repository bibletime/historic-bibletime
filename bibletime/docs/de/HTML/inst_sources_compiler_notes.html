3.3. Anmerkungen zum Compiler

Die zum kompilieren von BibleTime benötigte Zeit hängt von ihrer Systemgeschwindigkeit und der Menge des installierten Arbeitsspeichers ab. Wenn Sie ein System mit viel RAM haben, können Sie die Kompilierung durch Benutzung von "make -j2" beschleunigen. -j2 erlaubt es, dass zwei Prozesse zur gleichen Zeit gestartet werden.

Wenn ihr Rechner wenig Ram und einen langsamen Prozessor hat, ist die beste Art die Kompilierung zu beschleunigen X herunterzufahren und den Compiler von einem Terminal zu starten. Auf einem AMD 133 mit 32 MB Ram spart dies mehrere Stunden.
