Kapitel 2. BibleTime starten

BibleTime ist ein KDE Programm und ist in den KDE Arbeitsplatz integriert. Sie sollten ein Symbol um BibleTime zu starten auf dem Arbeitsplatz finden.

BibleTime kann aus anderen Fenstermanagern heraus aufgerufen werden (wie z.B. gnome) vorrausgesetzt, dass KDE installiert ist. Sie müssen ihr eigenes Symbol installieren umd BibleTime zu starten. Lesen Sie die Fenstermanager Hilfedateien um zu wissen wie.

BibleTime kann auch von einer Terminal Eingabeaufforderung aufgerufen werden. Um BibleTime aufzurufen öffnen Sie ein Terminal (jedes beliebige XTerminal) und geben sie folgendes ein: bibletime.

2.1. Startsequenz

Wenn BibleTime gestartet wird könnten Sie die folgenden Fenster sehen:

  • Keine Module installiert - Dies wird angezeigt, wenn Sie vergessen haben Sword Module zu installieren, oder wenn die Sword Module nicht gefunden werden konnten. Klicken Sie hier um die Keine Module installiert Seite zu sehen

  • Erster Start - Dies wird angezeigt, wenn BibleTime zum ersten Male aufgerufen wurde. Klicken Sie hier um die Erster Start Seite zu sehen

  • Tägliche Tipps - Sie werden bei jedem Start angezeigt, es sei denn Sie schalten sie ab.
