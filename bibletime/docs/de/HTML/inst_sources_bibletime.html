3.2. Installation von BibleTimes Quellcode

Als Erstes müssen Sie die heruntergeladene .tar.gz Datei in ein temporäres Verzeichnis entpacken (Wenn Sie die CVS Version benutzen, ist ihr Quellcode bereits entpackt, so dass Sie diesen Schritt auslassen können). Machen Sie ungefähr Folgendes:

 mkdir /tmp
 cd /tmp
 tar -xzf bibletime.tar.gz
 cd /tmp/bibletime

(Die tar Datei wird in ein Verzeichnis ähnlich wie "bibletime" entpackt werden)

Öffnen Sie nun ein Terminal und geben Sie die folgenden Kommandos ein, um den Quelltext zu kompileren.

 $ cd /tmp/bibletime         # Pfad des Quelltextes
 $ export QTDIR=/usr/lib/qt2   # QT 2.2.x Verzeichnis (wo QT 2.2 oder neuer installiert ist)
 $ export KDEDIR=/usr/local/kde2    # KDE2 Verzeichnis (wo KDE2 installiert ist)
 $ make -f Makefile.cvs             # nur für die CVS Version benötigt
 $ ./configure                      # erzeuge die Makefiles
 $ make                             # kompilere BibleTime
 $ su
 $ password *****
 $ make install                     # installiere BibleTime

Die Pfade für QT und KDE2 können auf ihrem Computer unterschiedlich sein.

Öffnen Sie nun ein Termianl als normaler Benutzer und tippen Sie "bibletime" ein. BibleTime startet.
