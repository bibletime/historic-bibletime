Kapitel 4. Das Modul ist verschlüsselt

Das von ihnen ausgewählte Modul ist verschlüsselt!

Diese Seite beschreibt was Verschlüsselung für Sie bedeutet.

4.1. Was ist ein verschlüsseltes Modul?
4.2. Wie kann ich ein Modul aufschließen?
4.3. Warum sind Module verschlossen?

4.1. Was ist ein verschlüsseltes Modul?

Ein verschlüsseltes Modul (Dokument) ist ein Modul dass mit einem Schlüssel "verschlossen" ist. Nur der Benutzer mit dem passenden Schlüssel kann es aufschließen und lesen.

4.2. Wie kann ich ein Modul aufschließen?

Für normale Benutzer ist es traurigerweise zu diesem Zeitpunkt nicht möglich verschlüsslte Module zu benutzen. Das Problem liegt darin, dass die Copyrightbesitzer der Module das Material nicht mit Sword veröffentlichen möchten. Aber falls Sie den passenden Schlüssel haben klicken Sie mit der rechten Maustaste auf das Modul und wählen dann "Modul aufschließen...". Ein Dialog erscheint, geben Sie hier den Schlüssel ein um das verschlüsselte Modul aufzuschließen.

4.3. Warum sind Module verschlossen?

Module sind verschlossen weil die Copyrightinhaber der Module ihre Texte nicht mit durch das Sword Projekt veröffentlichen möchten. Bitte kontaktieren Sie die Copyrightinhaber und fragen Sie danach, die Module verfügbar zu machen. Falls Sie dafür bezahlen möchten, sagen Sie dies, dies ist eine Sache die sie mögen. Wir wissen dass freie Module toll wären, deshalb probieren Sie diese zu bekommen. Sie können ebenfalls Crosswire besuchen und die Informationen über die Anstrengungen lesen, die unternommen wurden die Texte durch Crosswire zu veröffentlichen. Vielen Dank für ihre Mühen!
