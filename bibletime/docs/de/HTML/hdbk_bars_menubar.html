4.2.3. Settings

Zeige Werkzeugleiste

Zeige oder verstecke die Hauptwerkzeugleiste, normalerweise wird sie angezeigt.

Konfiguriere Werkzeugleiste

Zeigt den Konfiguriere Werkzeugleiste... Dialog an.

Einstellungen

Zeigt den Konfigurationsdialog an

Zeige Strongs Nummern

Zeige Strongs Nummern in den texten an (nur bestimmte Module benutzen diese Funktion).

Zeige Fußnoten

Zeige Fußnoten in den texten an (nur bestimme Module benutzen diese Funktion)

Zeige Hauptgruppe

Zeigt oder versteckt die Hauptgruppe. Normalerweise ist sie nicht versteckt, es sei denn Sie möchten den größt möglichen Platz zum Anzeigen der Module. BTW, durch Verkleinern der Hauptgruppe kann man dass Gleiche erreichen, jedoch zu einem geringeren Ausmaß.
