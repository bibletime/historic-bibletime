2.2. Installation der benötigten Programme

Laden Sie nun die benötigten Pakete herunter und kompilieren Sie die Bibliotheken

Als aller Erstes müssen sie die QT 2 Quellcodes kompilieren, wenn QT 2.2 oder neuer nicht mit ihren Installationsmedien geliefert wurde. Wie nehmen an, dass der Benutzer den Quellcode von QT in /usr/lib/qt2 hat, und dass sein KDE2 Verzeichnis /usr/local/kde2 ist. Wenn Sie die Werte nicht mögen, benutzen Sie Ihre eigenen.

 export QTDIR=/usr/lib/qt2
 cd $QTDIR
 ./configure -sm -system-libpng -system-jpeg
 make

Wenn Sie genügend Arbeitsspeicher und einen schnellen Computer haben können Sie den Kompilierungsvorgang durch das Hinzufügen von "-j2" zu den make Optionen beschleunigen. Zum Beispiel "make -j2". Wenn Sie einen langsamen Computer und wenig Arbeitsspeicher haben, schließen Sie X Windows und starten Sie make von einem Terminal aus um den Vorgang zu beschleunigen. Nachdem der Übersetzungsvorgang beendet ist, müssen Sie ihre KDE2 Quellcodes übersetzen. Machen Sie in etwa folgendes:

 export QTDIR=/usr/lib/qt2
 export KDEDIR=/usr/local/kde2
 cd /usr/src/kde2/kdelibs
 ./configure
 make
 make install

Die Compileirung von QT und KDE sollte ohne Fehler beendet werden. Es ist wieder möglich "-j2" zu benutzen um dem Compilierungsprozess zu beschleunigen, wenn Sie einen schnellen PC haben.
