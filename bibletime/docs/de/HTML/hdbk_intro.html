Kapitel 1. Einleitung

BibleTime ist ein mächtiges Bibelstudien Werkzeug, geschrieben für die KDE Umgebung auf dem Linux Betriebssystem. Es basiert auf der Sword Bibliothek, welche die Back-End Funktionalität für BibleTime zur Verfügung stellt, zum Beispiel Bibeltexte anschauen, Suchen, Lesezeichen anlegen usw. Sword ist das Flagschiffprodukt der Crosswire Bibelgesellschaft.

Es gibt drei Typen von Modulen, die angezeigt werden können:

Bibeln

Der ganze Text, mit optionalen Strong's Nummern und Fußnoten im Text

Kommentare

Mit einem „Persönlichem“ Kommentar können Sie ihre eigenen persönlichen Notizen zu verschiedenen Bereichen der Bibel anlegen.

Lexika

Für Wortdefinitionen und Bibelthemen. Sie können einen Schlüssel aus einer Liste auswählen der dann angezeigt wird.
