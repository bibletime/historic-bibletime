3.5. Ausgabe

3.5.1. Exportieren

Wie man Text als HTML bzw. Text exportieren oder in die Zwischenablage kopiert. Machen Sie einen Mausklick mit der rechten Taste in einem Dokument aus und ein Menü erscheint. Wählen Sie die gewünschte Aktion aus. Eine Box zum Abspeichern wird erscheinen, in der sie den gewünschten Dateinamen eingeben können.

Informationen zum Kontextmenü werden hier später hinzugefügt.

3.5.2. Drucken

Das Drucksystem wurde verändert. Dieser Abschnitt befindet sich in der Überarbeitung.

Drucken innerhalb von BibleTime ist ziemlich rudimentär und ist als Hilfe gedacht. Wenn Sie ein Dokument oder eine Präsentation mit Hilfe von Texten in BibleTime zusammenstellen, so empfehlen wir ihnen eines der Werkzeuge ihres Systems dafür zu verwenden. Lesen Sie auch die Informationen über das Exportieren.

Um zu drucken, wählen Sie ein geöffnetes Dokument aus und wählen anschließend Drucken aus. Der Druckerdialog erscheint. Sie haben die Wahl zwischen dem Drucken auf den Drucker oder der Vorschau mit einem Programm wie kghostview.

Kontextmenü Informationen kommen später.
