Antwort
Was ist ein verschlüsseltes Modul? Ein verschlüsseltes Modul ist ein Modul, welches mit einer Zeichenfolge verschlüsselt wurde. Nur Nutzer mit dem passenden Schüssel sind in der Lage, das Modul zu lesen.
Wie kann ich ein Modul entschlüsseln? Für normale Nutzer ist es im Moment leider unmöglich, verschlüsselte Module zu nutzen. Das Problem ist, dass die Copyrightinhaber der jeweiligen Module ihr Modul nicht zusammen mit SWORD herausgeben wollen.
Aber wenn Sie dennoch einen Schlüssel haben, klicken Sie mit der rechten Maustaste auf das Modul und wählen Sie "Modul entschlüsseln" Ein Dialog erscheint, bitte gebe Sie nun den zum Modul passenden Schlüssel ein, um das Modul zu entschlüsseln.
Warum Sind Module verschlüsselt? Module sind verschlüsselt, weil die Copyrightinhaber der jeweiligen Module ihr Modul nicht zusammen mit SWORD herausgeben wollen.
Bitte kontaktieren Sie die Copyrightinhaber und bitte Sie um die Module. Wenn Sie für die Module bezahlen würden, lassen Sie das die Inhaber wissen!! Wir wissen das frei verfügbare Module großartig wären, also versuchen Sie, sie zu bekommen!
