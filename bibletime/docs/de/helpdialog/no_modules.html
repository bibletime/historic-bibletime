Es sind keine SWORD Module installiert!

Frage

Antwort
Was ist ein Modul? Ein SWORD Modul ist die Basis von BibleTime. Ein Module kann eine Bible, ein Kommentar oder ein Lexicon sein.
Später werden wir Karten und andere interessante Sachen haben. Es ist unmöglich, mit BibleTime ohne installierte Module zu arbeiten, obwohl man BibleTime ohne installierte Module testen kann.
Wie löse ich dieses Problem? Bitte laden Sie einige Module herunter und installieren Sie sie!
Sie erhalten die Module auf www.crosswire.org. Bitte lesen sie die Installationsanleitung in unserem Handbuch und auf unserer Homepage.
Was soll ich tun wenn es nicht funktioniert? BittePlease lesen Sie die Dokumentation auf www.bibletime.de, wenn Sie es dennocht nicht schaffen, senden Sie eine email an info@bibletime.de.
