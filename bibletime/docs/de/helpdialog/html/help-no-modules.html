6.1 Was ist ein Modul?

Ein Sword Module ist die Basis von BibleTime. Ein Modul kann eine Bibel, ein Kommentar oder ein Lexicon sein. Später werden wir Karten und andere interessante Dinge haben. Es ist nicht möglich mit BibleTime zu arbeiten, wenn keine Module installiert sind, obwohl es möglich ist BibleTime ohne installiert Module zu testen. Wenn Module in BibleTime angezeigt werden, nennen wir sie Dokumente.

6.2 Wie löse ich das Problem?

Bitte laden sie ein paar Module herunter und installieren Sie diese. Sie können Module auf www.crosswire.org bekommen. Bitte lesen Sie die Installationsanweisungen in unserem Handbuch und auf unserer Homepage.

6.3 Was soll ich machen, falls es nicht funktioniert?

Bitte lesen Sie die Dokumentation auf www.bibletime.de und falls Sie es nicht zu funktionieren bekommen, schicken Sie eine eMail an info@bibletime.de .
