Es gibt kein mods.d Verzeichnis!

Frage

Antwort
Was ist das mods.d Verzeichnis? Das mods.d Verzeichnis ist das Verzeichnis, welches die Konfigurationsdateien der Module enthalten sollte! Sie benötigen dieses Verzeichnis, sonst können Sie BibleTime und SWORD nicht nutzen!
Bitte lesen Sie die Installationsanleitung auf unserer Homepage!
Wie löse ich dieses Problem? Die Lösung ist, ein mods.d Verzeichnis zu erstellen: Bitte gehen Sie in Ihr SWORD Verzeichnis und geben Sie "mkdir mods.d" ein, um das Verzeichnis zu erstellen. Nun kopieren Sie die Konfigurationsdateien Ihrer Module in dieses Verzeichnis unt löschen sie (falls vorhanden) die Datei "mods.conf", die nicht mehr benötigt wird. Das mods.d Verzeichnis ersetzt diese Datei.
Kann Ich BibleTime ohne dieses Verzeichnis starten? Ja, das ist möglich. Aber es ist nicht sehr praktisch. Es gibt im Groupmanager keine Module. So kann man BibleTime testen, aber mit BibleTime kann man nicht ohne Module arbeiten.
