Chapter 2. Beginnen Bibletime

BibleTime ist ein KDE Programm und ist in den KDE Arbeitsplatz integriert. Sie sollten ein Symbol für den Start von BibleTime auf dem Arbeitsplatz finden.

BibleTime kann auch aus einer Terminal Eingabeaufforderung heraus aufgerufen werden. Um BibleTime aufzurufen öffnen Sie ein Terminal (jedes beliebige XTerminal) und geben sie folgendes ein: bibletime.

Startsequenz

Wenn BibleTime gestartet wird, kann es vorkommen, dass Sie die folgenden Fenster sehen:

Bibletime:

  • Nein Informationen zu einzelnen Modulen Installierte ist bis Installieren Sword oder der/die Sword.

  • Erster Zeit Start des Programms ist der/die Zeit Bibletime ist.
