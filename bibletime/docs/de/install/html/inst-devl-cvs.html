CVS-Zugriff für Entwickler mit SSH

Nur Entwickler des Projektes haben mit dieser Methode Zugriff auf den CVS Baum. SSH1 muss auf ihrem Rechner installiert sein. Ersetzen Sie Modulname und Entwicklername mit den richtigen Werten. Geben Sie Ihr Passwort ein, wenn Sie dazu aufgefordert werden.

Wie man eine frische Kopie von BibleTime bekommt

cd $home/ihr Enwicklungsverzeichnis
export CVS_RSH=ssh      # Die Aktivierung von ssh ist nur einmal pro Sitzung erforderlich
cvs -z3 -d:ext:Entwicklernamename@cvs.bibletime.sourceforge.net:/cvsroot/bibletime co Modulename

Wie Sie Ihre Kopie von BibleTime aktualisieren

cvs -z5 update -Pd

Wie Sie Ihre Änderungen in das CVS Repository hochladen

cvs -z5 commit -m "Kommentare für diesen Commit"

Wie Sie neue Dateien hinzufügen, die Sie erzeugt haben

cvs -z5 add meine_neue_dateien
