Chapter 5. Zugang zum Entwickler-Quellcode

Anonymer CVS Zugriff

Das SourceForge CVS respository dieses Projekts kann man via anonymous (pserver) CVS auslesen, wenn man die folgenden Anweisungen befolgt. Das Modul, das Sie auslesen wollen, müssen Sie mit Modulename angeben. Bei der Aufforderung, ein Passwort einzugeben, drücken Sie einfach Enter. 

cvs -d:pserver:anonymous@cvs.bibletime.sourceforge.net:/cvsroot/bibletime login
cvs -z3 -d:pserver:anonymous@cvs.bibletime.sourceforge.net:/cvsroot/bibletime co Modulname

Updates innerhalb eines Verzeichnisses des Modules benötigen den -d Parameter nicht.
